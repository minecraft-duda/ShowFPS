# ShowFPS

[中文README]("https://github.com/minecraft-duda/ShowFPS/blob/main/README.md")

The developer has given up Forge,content is for Fabric ony!

This is a mod that displays the frame rate in the top-left corner.

This mod is very lightweight and has almost zero impact on FPS.

The FPS color changes with your frame rate: red when FPS < 60, green when FPS ≥ 60 and FPS ≤ 120, and blue when FPS > 120.

Press N (Default, changeable) to open the settings page!
